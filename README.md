# PDF Connection

PDF Connection은 여러 개의 PDF 파일을 원하는 순서대로 병합하고, 중복 페이지를 감지하여 자동으로 제거하며, 특정 페이지를 선택하여 손쉽게 제외할 수 있는 데스크톱 GUI 애플리케이션입니다.

---

## 주요 기능

1. **순서 지정 병합**
   - 병합할 PDF 파일들을 추가하고 `Move Up` / `Move Down` 버튼을 클릭하여 원하는 연결 순서를 자유롭게 조정할 수 있습니다.
2. **시각적 중복 페이지 제거 (Deduplication)**
   - 각 페이지를 시각적 이미지로 변환(72 DPI)하고 픽셀 데이터의 MD5 해시를 비교하여 완전히 동일한 내용을 갖는 페이지는 첫 페이지만 남기고 자동으로 제거합니다. 스캔본과 텍스트본 모두 완벽하게 감지합니다.
3. **특정 페이지 제외 (Exclusion)**
   - 파일별로 제외하고 싶은 페이지 번호나 범위를 1-based index 형태로 지정할 수 있습니다.
   - 예: `1` (1페이지 제외), `1, 3` (1, 3페이지 제외), `1-3, 5` (1~3페이지 및 5페이지 제외)

## 프로그램 실행 워크플로우

### 1단계: 설치 및 환경 구축
본 프로그램은 Python 3 및 `PyMuPDF` 라이브러리를 사용합니다. 아래 명령어로 필요한 패키지를 설치합니다.

```bash
# 가상환경 생성 및 활성화
python -m venv .venv
.venv\Scripts\activate   # Windows

# 의존성 패키지 설치
pip install pymupdf
```

### 2단계: 프로그램 실행
```bash
python main.py
```

### 3단계: PDF 파일 추가 및 순서 설정
1. **[Add PDF]** 버튼을 눌러 결합할 PDF 파일들을 목록에 추가합니다.
2. 목록에서 파일을 클릭하고 **[Move Up]** / **[Move Down]** 버튼으로 병합할 파일들의 순서를 변경합니다.

### 4단계: 제외할 페이지 지정 (선택 사항)
1. 목록에서 특정 파일을 클릭하면 우측 **"Page Exclusion Settings"** 패널에 파일 정보와 총 페이지 수가 표시됩니다.
2. **"Exclude Pages"** 입력 필드에 제외할 페이지 번호 또는 범위를 콤마(,)와 대시(-)로 입력합니다. (예: `1, 3-5`)
3. **[Apply Exclusions]** 버튼을 클릭하여 설정을 저장합니다.

### 5단계: 중복 페이지 제거 옵션 설정
- 하단의 **"Remove duplicate pages"** 체크박스가 활성화되어 있으면, 병합 시 중복되는 페이지가 1개만 남고 모두 자동으로 제거됩니다.

### 6단계: 병합 및 저장
- **[Merge PDFs]** 버튼을 누르고 저장할 위치와 파일명을 설정하면 병합이 완료됩니다.

## 프로젝트 구조

- `pdf_processor.py`: PDF 병합 및 중복/제외 페이지를 관리하는 핵심 비즈니스 로직
- `gui.py`: Tkinter 프레임워크를 사용한 데스크톱 그래픽 사용자 인터페이스
- `main.py`: 프로그램 진입점(Entry Point)
- `tests/test_pdf_processor.py`: 핵심 로직을 테스트하기 위한 자동 검수 유닛 테스트
